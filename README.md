# ThinkGISLayerTemplates
Layer templates (.tmltemplates) for use with creating new layers in ThinkGIS (WTH Technology Inc.) The templates are create from our award winning layers.

Place the layer template file of your choice into your LayerTemplates folder. Commonly found at "C:\ThinkGIS\LayerTemplates"

In Layer Settings, Click "Add Layer" Button -> "Create New Layer" -> Select the Layer Template from the list of templates. Note: You may need to restart ThinkGIS to see the new layer template in the list.
